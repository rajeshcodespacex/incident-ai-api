## About This Project

In enterprise IT environments, support engineers manually analyze incident tickets, identify root causes, and write resolution steps — a process that takes significant time per ticket. This API automates that first-level diagnosis using Google Gemini AI, instantly providing root cause analysis, step-by-step resolution guidance, and severity classification.

**Real background:** Built based on 2.6 years of hands-on experience resolving INC and RITM tickets in ServiceNow, specializing in Globalscape EFT secure file transfer systems.

## Tech Stack

- **FastAPI** — Python web framework for building REST APIs
- **Google Gemini AI** — AI-powered incident analysis and resolution
- **PostgreSQL** — Production database for incident storage
- **SQLAlchemy** — ORM for database operations
- **JWT Authentication** — Secure token-based authentication
- **Passlib + Bcrypt** — Password hashing
- **Pytest** — Automated testing
- **Uvicorn** — ASGI server

## Features

- JWT-based user authentication (register and login)
- AI-powered incident analysis using Google Gemini
- Automatic root cause identification from incident description
- Step-by-step resolution suggestions generated by AI
- Automatic severity classification (HIGH, MEDIUM, LOW)
- Complete incident lifecycle management (create, view, update, delete)
- Dashboard with real-time incident statistics
- Role-based access control (admin vs user)
- Admin panel to manage all incidents and users
- 8 automated tests covering auth and API endpoints
